This guide is provided to help users understand how monitoring the infrastructure with **Automium** and configure specific alerts to be notified of system failures and performance issues.
  
The guide contains the following information:  
* Instructions for creating a **[Prometheus](https://www.prometheus.io)** server.  
* An explanation on how Prometheus can be configured for **Monitoring** all the Infrastructure services.  
* An overview of the features and functionality of Prometheus for **Alerting**.  
* Instructions for creating custom **Alerts**.  

Let's login into the **Automium Dashboard** to create the [Prometheus Server](prometheus).
